## Getting Started
1. Clone the repo
`https://github.com/ferris110443/05.Login_TodoList_Flask.git`
2. Activate virtual environment 
3. Install packages
`pip install -r requirements.txt`
5. Modify SQLALCHEMY_DATABASE_URI in __init__.py file
6. Start
